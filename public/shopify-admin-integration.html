Add Reseller Selection Directly to Product Editing Pages

üéØ Goal

Add a "Select Resellers" section directly inside each product editor in Shopify admin, so you can assign resellers to products without leaving the product editing page.

Step 1: Add the Script to Your Shopify Theme

Add this script to your Shopify admin theme to inject the reseller selection interface:

<script src="https://your-app-url.railway.app/product-reseller-embed.js"></script>

Note: Replace your-app-url.railway.app with your actual Railway deployment URL.

Step 2: Add to Admin Theme (admin.liquid)

Add the script to your admin theme's admin.liquid file:

<!-- Add this before the closing </body> tag --> <script src="https://your-app-url.railway.app/product-reseller-embed.js"></script>

Step 3: Alternative - Browser Extension

For immediate testing, you can use a browser extension to inject the script:

// Create a browser extension with this content script: // manifest.json { "manifest_version": 3, "name": "Reseller Selection", "version": "1.0", "content_scripts": [{ "matches": ["*://*.myshopify.com/admin/products/*"], "js": ["https://your-app-url.railway.app/product-reseller-embed.js"] }] }

Step 4: Manual Injection (For Testing)

To test immediately, open the browser console on any product editing page and run:

// Copy and paste this into the browser console: const script = document.createElement('script'); script.src = 'https://your-app-url.railway.app/product-reseller-embed.js'; document.head.appendChild(script);
