üöÄ How to Integrate Reseller Selection

Follow these steps to add reseller selection functionality to your Shopify product pages:

1. Add the JavaScript Integration

Include the product integration script in your product template:

<script src="/product-integration.js"></script>

2. Add to Product Template

Add this code to your product.liquid template or create a new section:

{% comment %} Reseller Selection Section {% endcomment %} <div id="reseller-section"> <!-- The reseller selection UI will be automatically inserted here --> </div>

3. Customize the Integration

Modify the createResellerSection() function in product-integration.js to customize where the reseller section appears on your page.

4. Test the Integration

Visit any product page to see the reseller selection in action. The system will automatically:

  • Load all available resellers
  • Show currently associated resellers
  • Allow selection of multiple resellers
  • Save changes automatically
